# Django Sticky Notes App

A lightweight, full-stack web application designed for creating, viewing, updating, and deleting digital sticky notes. This project demonstrates a clean, decoupled MVC architecture using Django and SQLite.

## 🚀 Features
* **Full CRUD Functionality:** Create, Read, Update, and Delete notes seamlessly from the interface.
* **Automated Test Suite:** Includes comprehensive backend unit tests verifying all database and routing operations.
* **Clean Architecture:** Well-organized project separation keeping framework configurations distinct from application logic.

## 🛠️ Tech Stack
* **Backend:** Python, Django
* **Database:** SQLite
* **Frontend:** HTML5, CSS3
